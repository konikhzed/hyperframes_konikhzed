# HyperFrames Konikhzed Edition 🔥

**Write HTML → Render Golden Videos**

فورک شخصی از [HyperFrames](https://github.com/heygen-com/hyperframes) با تم **نسبت طلایی (Golden Ratio)**، اعداد Konikhzed و ادغام کامل با پروژه‌های من:

- [SVG-konikhzed](https://github.com/konikhzed/SVG-konikhzed)
- [Konikhzed_AI](https://github.com/konikhzed/Konikhzed_AI)
- [Fayton & Fayton-Weather](https://github.com/konikhzed/Fayton-Weather)

## ویژگی‌های خاص Konikhzed
- انیمیشن‌های مبتنی بر φ = 1.6180339887
- الگوهای SVG لایه‌ای با اعداد Konikhzed (16.3248, 8.1624, ...)
- visualization آب‌وهوا با Golden Uncertainty
- deterministic rendering برای تحقیقات علمی/AI

## نصب سریع
```bash
npx hyperframes init konikhzed-video --template konikhzed-golden
```
